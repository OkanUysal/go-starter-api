# my-api

A Go API generated with go-starter.

## Features

- auth
- logger
- response

## Getting Started

```bash
go mod tidy
cp .env.example .env
go run main.go
```

## API Endpoints

- `GET /health` - Health check
- `GET /api/v1/users` - Get users

## Deployment

This project is ready for Railway deployment.

1. Push to GitHub
2. Connect to Railway
3. Deploy!
